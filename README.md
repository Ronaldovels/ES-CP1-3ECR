# FiaPals

## 📌 Descrição do Problema
Muitos alunos da FIAP sentem dificuldade em se conectar com colegas de curso, compartilhar interesses, organizar eventos e participar ativamente da vida acadêmica. A ausência de uma rede social interna limita a integração e o engajamento entre os estudantes.

## 🚀 Solução Proposta
O FiaPals é uma rede social exclusiva para alunos da FIAP, permitindo criar perfis, interagir com a turma, publicar posts, organizar e participar de eventos, além de explorar outras turmas e atividades. A plataforma incentiva a colaboração, networking e participação em eventos acadêmicos.

## 🛠️ Tecnologias Utilizadas
- Python 3.x
- Armazenamento em JSON (persistência de dados)
- Bibliotecas padrão: os, json, hashlib, uuid, datetime

## ⚙️ Como Executar
1. Certifique-se de ter o Python 3 instalado.
2. Baixe/clique no arquivo `fiapals.py`.
3. Execute no terminal:
	```
	python fiapals.py
	```
4. Siga as instruções no menu para criar uma conta ou fazer login.

## 📂 Estrutura do Projeto
```
fiapals.py         # Código principal da aplicação
fiapals_db.json    # Banco de dados gerado automaticamente (JSON)
README.md          # Documentação do projeto
```

## 🧩 Funcionalidades Implementadas
- Cadastro e login de usuários
- Edição de perfil (bio e interesses)
- Visualização e interação com a turma
- Publicação de posts no feed da turma
- Criação e inscrição em eventos (abertos ou restritos)
- Exploração de turmas e eventos de toda a FIAP
