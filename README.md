# FiaPals

## 📌 Descrição do Problema
Muitos alunos da FIAP sentem dificuldade em se conectar com colegas de curso, compartilhar interesses, organizar eventos e participar ativamente da vida acadêmica. A ausência de uma rede social interna limita a integração e o engajamento entre os estudantes.

## 🚀 Solução Proposta
O FiaPals é uma rede social exclusiva para alunos da FIAP, permitindo criar perfis, interagir com a turma, publicar posts, organizar e participar de eventos, além de explorar outras turmas e atividades. A plataforma incentiva a colaboração, o networking e a participação em eventos acadêmicos.

## 🛠️ Tecnologias Utilizadas
- Python 3.x
- Flask
- Flask-CORS
- JSON para persistência de dados
- HTML, CSS e JavaScript
- Upload de imagens

## ⚙️ Como Executar
1. Certifique-se de ter o Python 3 instalado.
2. Instale as dependências necessárias para executar o projeto.
3. Execute o arquivo principal da aplicação.
4. Acesse a aplicação pelo navegador no endereço local configurado.

## 📂 Estrutura do Projeto
- `fiapals.py` — arquivo principal do backend da aplicação
- `index.html` — interface principal da aplicação
- `fiapals_db.json` — banco de dados gerado automaticamente em JSON
- `uploads/` — diretório para armazenamento das imagens enviadas
- `README.md` — documentação do projeto

## 🧩 Funcionalidades Implementadas
- Cadastro e login de usuários
- Edição de perfil com bio e interesses
- Visualização da turma do usuário
- Publicação de posts com texto e imagem
- Feed da turma
- Feed geral com posts de pessoas seguidas
- Sistema de seguir outros usuários
- Criação de eventos
- Inscrição em eventos
- Eventos abertos, por curso ou restritos à turma
- Controle de vagas, pagamento, idade mínima e confirmação de presença
- Exploração de turmas e membros
